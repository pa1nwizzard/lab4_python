# Лабораторная работа №4
> Замыкания

## Задание
> Замыкание-калькулятор, накапливающее результат и поддерживающее 4 арифметичесике операции.
> Декоратор, который будет запускать функцию указанное число раз с указанными параметрами и возвращать последвательность результатов.

## Ход выполнения работы

* Замыкание-калькулятор
Функция `make_calc(op, initial)` возвращает внутреннюю функцию `calc(x)`, которая сохраняет текущий результат и счётчик вызовов через `nonlocal`. После 4 вызовов результат замораживается.

* Декоратор-функция `repeat_calls`
Принимает параметры `times` и `arguments`. При наличии списка аргументов подставляет их последовательно, иначе использует аргументы вызова. Возвращает список результатов.

* Класс-декоратор `RepeatCalls`
Аналог функции, но с более явной структурой: параметры хранятся в `__init__`, логика обёртки — в `__call__`.
